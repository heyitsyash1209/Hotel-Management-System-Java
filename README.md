# 🏨 YG Peacewood Hotel Management System

A terminal-based hotel booking system built using Java and OOP concepts.

## Features
- Room Booking
- Billing System
- Checkout & Cancellation
- Rating System
- CSV Data Storage

## Run Project

```bash
cd src
javac models/*.java services/*.java main/*.java
java main.HotelBooking
```
